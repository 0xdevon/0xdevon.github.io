3.使用Navicat生成表结构数据字典

  1. 在Navicat中“新建查询”，输入下列脚本
SELECT
	COLUMN_NAME 列名,
	COLUMN_TYPE 数据类型,
	DATA_TYPE 字段类型,
	CHARACTER_MAXIMUM_LENGTH 长度,
	IS_NULLABLE 是否为空,
	COLUMN_DEFAULT 默认值,
	COLUMN_COMMENT 备注
FROM
	INFORMATION_SCHEMA. COLUMNS
WHERE
	-- super_express为数据库名称，到时候只需要修改成你要导出表结构的数据库即可
	table_schema = 'super_express'
AND 
	-- s_patient_list为表名，到时候换成你要导出的表的名称
	table_name = 'example'

  1. 全选输出的查询结果，右键选择复制为“制表符分隔值（字段名和数据）”

  2. 在Excel中粘贴上面复制的制表符分隔值，再在菜单栏选择添加“所有框线”，选择需要的部分复制

  3. 在需要粘贴的Word文档中选择“使用目标样式”

4.拉取Git远程仓库dev分支代码合并到本地dev分支以及自己的分支步骤（如zer09b）

  1. 在本地自己的分支下（如zer09b）下，需要完成以下步骤：
1.git add .
2.git commit -m"备注"
  1. 切换到本地dev分支，需要完成的步骤：
1.git switch dev
2.git pull
3.git add .
4.git commit -m"备注"
5.git merge zer09b
6.git add .
7.git commit -m"备注"
  1. 继续切换到本地自己的分支下（如zer09b）,需要完成以下步骤：
1.git switch zer09b
2.git merge dev
3.git add .
4.git commit -m"备注"
5.git push

5.用IDM下载百度网盘文件

  1. Google Chrome Canary 打开分享链接
  2. 转存到我的网盘
  3. 在网盘中点击转存的文件，生成链接
  4. 将应用ID修改为778750，关闭对话框
  5. 再次生成链接，右键单击链接，使用IDM下载即可

6.使用maven构建web应用

  1. 在IDEA中“Maven Projects”的Lifecylce文件夹下双击“package”，项目将自动被打包成为一个jar包（位置位于项目target文件夹下）

  2. 在WindowsPowerShell中定位jar包绝对路径，使用下方命令编译运行上述已经打包完成的jar包：

    例如："F:\**\**\target> java -jar .\jar包名称(例如：helloworld-0.0.1-SNAPSHOT.jar)"

7.Windows10常用快捷键总结

  1. 打开设置：Windows+I

  2. 打开文件资源管理器：Windows+E

  3. 启动任务栏固定的应用程序（从Windows徽标右侧第一个应用程序计数）：例如启动第一个应用程序则按Windows+1 ，以此类推

  4. 快捷自由截屏：Windows+Shift+S ；PC端微信正在运行的情况下，可按Alt+A 进行自由截屏；PC端QQ正在运行的情况下， 可以按Ctrl+Alt+A 进行自由截屏

  5. 快速打开系统投屏/拓展显示器选项：Windows+P

  6. 快速打开任务管理器：Ctrl+Shift+Esc （WindowsXP之后系统版本），之前常用的Ctrl+Alt+Delete 快捷键已经变成了高级启动选项快捷键

  7. 全屏截屏：Windows+PrtScr ；当前程序窗口截图：Ctrl+Alt+PrtScr

  8. 快速打开"运行"窗口Windows+R ，下面附上常见命令和功能：

    1.打开系统服务：services.msc
2.打开cmd窗口：cmd
3.打开系统配置：mscconfig
4.打开Windows恶意软件删除工具：mrt
5.同一局域网中设备间通过IP地址访问共享文件夹：//+被共享的文件所在设备IP
6.查看个人电脑详细配置/属性：msinfo32
7.打开系统注册表：regedit
8.打开控制面板中“程序和功能”：appwiz.cpl
9.查看自己的电脑最近有过哪些活动：recent
10.确定自己的电脑系统是否激活成功（正版系统）：slmgr.vbs -xpr
11.打开DirectX诊断工具：dxdiag
12.打开本地组策略编辑器：gpedit.msc

  9. 打开Windows通知中心：Windows+A

  10. 所有窗口最小化（不可复现）：Windows+M ；最小化/恢复当前所有窗口：Windows+D

  11. Alt+F4 ：快速（强制）关闭当前窗口，在没有窗口显示的情况下为"关机"快捷键

  12. 全选：Ctrl+A ；复制：Ctrl+C；粘贴：Ctrl+V；撤销：Ctrl+Z；重做：Ctrl+Y；剪切：Ctrl+X ；加粗：Ctrl+B；下划线：Ctrl+U；居中：Ctrl+E；字体倾斜：Ctrl+I；复制格式：Ctrl+Shift+C；粘贴格式：Ctrl+Shift+V ；分页符：Ctrl+Enter

  13. 打开Windows徽标右键菜单：Windows+X

  14. 快速打开WindowsPowerShell（管理员）：Windows+X -A ，快速打开WindowsPowerShell：Windows+X -I

  15. 保存：Ctrl+S；另存为：Ctrl+Shift+S
